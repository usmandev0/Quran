# Quran App

## Overview
A modern and clean Quran application inspired by high‑quality UI principles. The app includes Arabic text rendering, translations, audio playback, bookmarking, and smooth navigation.

### **Design Reference (Figma)**
This UI is based on the Figma community design:
**https://www.figma.com/community/file/966921639679380402**

---

## Features
- Arabic Quran text with proper typography
- Surah and Juz navigation
- Modern UI based on Figma design system
- Built with Jetpack Compose / Compose Multiplatform

---

## UI & Design Philosophy
The design follows:
- Minimal layout structure
- Smooth spacing and balanced typography
- Clear Surah list design
- Focused reading experience with comfortable margins
- Consistent color tokens and elevation

Use the Figma file to view:
- Typography scale (Arabic + Latin)
- Color palette
- Spacing system
- Surah list styles
- Reader screen layout
